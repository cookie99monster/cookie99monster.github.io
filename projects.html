Projects Showcase


A collection of selected work. I love building anything that's creative, impactful, elevates user experience and has a potential to change lives.

Accident Detection and Emergency reporting service

It is observed that the rate of deaths per thousand vehicles in 2021 has increased from 0.45 in 2020 to 0.53 in 2021. 4,03,116 road accidents caused 1,55,622 deaths and injuries to 3,71,884 persons during 2021. This project aims at detecting accident on dashcam using deep learning and computer vision(pytorch) and report it to nearby emergency services with valid accident images.

Sentiment Prediction: A transfer learning approach using BERT

One of the most critical and underutilized aspect of business and customer relationship is understanding the opinion and sentiment of customers. Opinion mining using sentiment models can be used to automatically extract and classify the opinions expressed in reviews or other types of text. This project aims at detecting the sentiment of a customer review using a sentiment model made from BERT and fine tuned on imdb movie reviews dataset.

Anime face generator using Generative Adversarial Networks (GANs)

GANs are an exciting and rapidly changing field, delivering on the promise of generative models in their ability to generate realistic examples across a range of problem domains, most notably in image-to-image translation tasks such as translating photos of summer to winter or day to night, and in generating photorealistic photos of objects, scenes, and people that even humans cannot tell are fake. This project aims to create anime faces using Generative Adversarial Networks (GANs).

Netflix Movie Title Recommendation using Annoy

Annoy (Approximate Nearest Neighbors Oh Yeah) is a C++ library with Python bindings to search for points in space that are close to a given query point. It also creates large read-only file-based data structures that are mmapped into memory so that many processes may share the same data. This project aims at creating a personalized movie recommendation system for Netflix using the Annoy library for approximate nearest neighbor searches.

Handwritten digits recognition: A NumPy only program

Handwriting recognition has always been a challenging task in the field of AI/ML mainly because it can vary widely from one person to another, and even within the same person. This makes it difficult for a handwriting recognition system to accurately interpret all types of handwriting. This project aims to tackle this problem by training a model written entirely from scratch only using NumPy and MNIST handwritten digits dataset.
